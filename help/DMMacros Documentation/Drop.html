MACRO Drop

Version: 1.01                                                                                                           
Author: Daniel Mastropietro                                                                                             
Created: 9-Feb-01                                                                                                       
Modified: 19-Jun-03

DESCRIPTION:

This macro drops variables from a data set without showing a warning or                                                                                          
error message in case any of the variables do not exist in the data set.

OTHER MACROS USED IN THIS MACRO:

- %ExistVar                                                                                                             
- %GetNroElements
